# Cat Prep — Introducing Cat to Baby

> **Start:** Week 24-26 (gradual introduction to changes)

## Before Baby Arrives

### Gradual Changes (Start Now)
- [ ] Start playing baby sounds (crying, cooing) at low volume, gradually increase
- [ ] Introduce baby gear (swing, bassinet) early so cat can investigate and get bored
- [ ] Set up baby's sleep space and establish it as off-limits
- [ ] If changing any cat routines (feeding spot, litter location), do it NOW — not after baby arrives
- [ ] Practice carrying a doll/blanket while interacting with cat

### Sleep Space Safety
- [ ] **Critical: Cat must not have access to baby's sleep space** — risk of suffocation
- [ ] Options:
  - Crib net/tent (controversial; some pediatricians advise against as entanglement risk)
  - Close bedroom door (need a monitor)
  - Screen door on nursery/bedroom
  - Keep cat out of bedroom entirely
- [ ] Test your chosen barrier before baby arrives

### Scent Introduction
- [ ] Bring home a blanket or hat baby wore in hospital before bringing baby home
- [ ] Let cat sniff and investigate
- [ ] Reward calm behavior

## When Baby Comes Home

- Greet the cat first (without baby) — one parent goes in ahead
- Let cat approach baby on their own terms; don't force interaction
- Supervise all interactions
- Maintain cat's routine as much as possible (feeding times, play, attention)
- Give cat "escape routes" — high perches, separate room

## Ongoing

- **Never leave cat and baby unsupervised** (until child is old enough to interact safely)
- Watch for stress signs in cat: hiding, appetite changes, litter box issues, aggression
- Secure cat food, water, and litter box away from baby once mobile (~6 months)
- Consider Feliway diffuser if cat seems stressed
