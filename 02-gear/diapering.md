# Diapering

> **Status:** Researching

## Cloth vs Disposable

| Factor | Disposable | Cloth |
|--------|-----------|-------|
| Convenience | Easy, no laundry | Requires washing routine |
| Cost (first year) | ~$800–$1,000 | ~$300–$500 upfront + laundry |
| Environmental | Landfill waste | Water/energy for washing |
| Apartment factor | Need diaper pail | Need storage + washer access |
| Our situation | Simpler for apartment life | Possible if in-unit laundry |

> **Decision:** TBD — discuss with partner

## Supplies Needed

### If Disposable
- [ ] Newborn diapers (1-2 packs — don't overbuy, may skip NB size)
- [ ] Size 1 diapers (stock up more on this size)
- [ ] Wipes (fragrance-free)
- [ ] Diaper pail (Ubbi recommended for small spaces — uses regular trash bags)
- [ ] Diaper cream (zinc oxide-based)
- [ ] Changing pad + 2-3 covers
- [ ] Portable changing pad for diaper bag

### If Cloth
- [ ] 24-36 cloth diapers
- [ ] Wet bags (2-3)
- [ ] Cloth-safe diaper cream
- [ ] Sprayer attachment for toilet
- [ ] Diaper-safe detergent
